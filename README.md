# Context-Aware Word Prediction for Social Media Data
● Built and optimized a context-aware word prediction system using a dataset of over 1 million tweets and comments.

● Implemented text preprocessing pipelines with NLTK for stop word removal and tokenization.

● Leveraged TF-IDF for feature extraction to identify key contextual terms across documents.

● Applied cosine similarity to analyze text relationships and improve context-based word prediction accuracy.

Dataset: https://www.kaggle.com/kazanova/sentiment140/data
